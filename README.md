# zigbee

Rust library implementing the ZigBee protocol.

## Work in progress

This library aims to provide an implementation of the Zigbee Cluster Library as defined in document `07-5123`, revision
`8`.

This library is a work in progress and is not yet ready for use.

## Documentation

- [Zigbee specification](https://zigbeealliance.org/wp-content/uploads/2021/10/07-5123-08-Zigbee-Cluster-Library.pdf)

## Legal

Tis library is free software and is not affiliated with the Zigbee Alliance or the Zigbee protocol. It is an independent
implementation of the Zigbee Cluster Library.
